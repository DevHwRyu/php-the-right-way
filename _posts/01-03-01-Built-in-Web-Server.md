## 내장 웹 서버 {#builtin_web_server_title}

웹서버를 따로 설치하는 부담 없이도 PHP 학습을 시작할 수 있습니다(PHP 5.4+ 필요). 웹서버를 실행하려면 아래와 같은 커맨드를 여러분의 프로젝트 폴더에서 실행하면 됩니다.

    > php -S localhost:8000

* [내장 웹 서버에 대해서 배우기][cli-server]
